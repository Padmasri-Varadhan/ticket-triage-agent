# Ticket Triage Agent

An AI-powered system that automatically classifies, prioritizes, and routes support tickets using NLP and Explainable AI.

## Features
- Ticket classification (Billing, Technical, Feature Request)
- Priority detection (High, Medium, Low)
- Sentiment analysis
- Explainable AI (reasoning behind decisions)

## Workflow
1. User submits ticket
2. AI analyzes text
3. Classifies category
4. Assigns priority
5. Generates explanation

## Tech Stack
- Frontend: React
- Backend: Node.js / Flask
- AI: NLP / LLM
